1. Map пен List айырмашылығы неде?

List:

* Элементтер тізім түрінде сақталады
* Индекс арқылы қол жеткіземіз (0,1,2…)
* Бірдей элементтер қайталана береді

Map:

* Деректер кілт → мән (key → value) түрінде сақталады
* Әр кілт бірегей (повтор болмайды)
* Кілт арқылы тез іздейміз

👉 Қысқасы:
List = тізім,
Map = сөздік (кілтпен жұмыс)

⸻

2. Қай кезде Map қолдану тиімді?

Map қолдану тиімді, егер:

* Деректі тез табу керек болса
* Әр элементтің бірегей кілті бар болса
* Индекспен емес, атаумен/ID арқылы іздеу керек болса

👉 Мысал:

* Тауар аты → тауар
* Студент ID → студент

⸻

3. CRUD деген не?

CRUD — деректермен жұмыс істеудің 4 негізгі операциясы:

* Create – жаңа дерек қосу
* Read – деректі көру/оқу
* Update – деректі өзгерту
* Delete – деректі жою
